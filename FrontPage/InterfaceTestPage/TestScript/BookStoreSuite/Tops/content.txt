!3 二、接口描述：
童书榜单接口!-
-!与主站对接，获取主站各种童书榜单

'''参数type解释'''：

| comment |
| bk_24hours_ts | 图书畅销榜24小时童书榜 |
| bk_recent7_ts | 图书畅销榜最近7天童书榜 |
| bk_week_ts | 图书畅销榜周榜童书榜 |
| bk_recent30_ts | 图书畅销榜最近30天童书榜 |
| bk_month_ts | 图书畅销榜月榜童书榜 |
| bk_halfyear_ts | 图书畅销榜半年榜童书榜 |
| bk_year_ts | 图书畅销榜年榜童书榜 |

!3 '''参数path解释'''：
| comment |
| 1 | 0～2 岁 |
| 2 | 3～6 岁 |
| 3 | 7～10 岁 |
| 4 | 11～14 岁 |
